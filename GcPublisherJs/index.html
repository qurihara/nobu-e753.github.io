About

Input Device Scanner for GameControllerizer. Control signals from a input device is scanned, then published as MQTT message.


Query parameters

ID Type Description
fps Int Device scanning fps (default=20)
gamepad String Gamepad model ( {"classic", "psx", "xbox", "custom"} )

Customize

Local hosting

For much better performance, GcPublisher can be executed on local http server such as npm-http-server. 
  $ npm install http-server
  $ git clone [this repository]
  $ cd [target dir]
  $ http-server

Custom gamepad

Edit `js/gc_gamepad_scanner/custom.js`, then reload page with query paramter (`?gamepad="custom"`).
